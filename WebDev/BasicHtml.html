Some HTML Tags

< h1, h2, h3, h4, h5, h6 -- Heading tag
begins a heading. Heading scope is determined by numbers 1-6
< /h1, /h2, /h3, /h4, /h6 -- Heading terminate tag
< heading termination tag. numbers correspond to the opening tags
< p> - paragraph'
begins a paragraph block
< /p> - paragraph teminate tag'
ends a paragraph block
< div >, < /div >
division tag, demarcates divions of html code for special processing.
< script >, < /script >
script tag -- allows a script to be processed in the loading of the html page. the script code can be specified within the script and /script tags.
